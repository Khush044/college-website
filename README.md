# College Website Portal
# Description

This is a full-stack web application developed using Node.js, Express, MongoDB, and EJS.

It allows users to:

Register and login
Explore available courses
Apply for programs
View course details
Submit applications

# Features

User authentication system (Login & Signup)
Role-based access (Student / Faculty / Admin)
Course listing with details
Apply Now functionality with form submission
Application data stored in MongoDB
Responsive and modern UI
Basic dashboard after login
Form validation and error handling

# Tech Stack

Node.js
Express.js
MongoDB
EJS (Templating Engine)
HTML, CSS, JavaScript

# How to Run
Clone the repo:
git clone https://github.com/Khush044/college-website.git
Install dependencies:
npm install
Add .env file:
PORT=4000
MONGODB_URI=your_mongodb_connection_string
Run server:
npm start
Open browser:
http://localhost:4000
